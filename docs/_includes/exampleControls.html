Olli controls

Press t to jump to the Olli structure.

When Olli has focus, the following keyboard commands are available:

  • Up / Down arrow — move up or down a level in the tree view
  • Left / Right arrow — move to previous or next item in a level
  • Home / End — move to first or last item in a level
  • x — jump to x-axis of current chart
  • y — jump to y-axis of current chart
  • l — jump to legend of current chart
  • t — jump to top level of current tree

Customization controls

The settings menu and command box allow you to make changes to the appearance of individual tokens. You can access them with these keyboard commands:

  • m / Escape — open or close the settings menu
  • i / Escape — open or close the command box

This is the list of tokens:

  • index — index in set (e.g. "1 of 5 ")
  • type — type of element (e.g. "temporal scale", "line chart")
  • size — size of children (e.g. "10 values")
  • data — data value or range (e.g. "0 to 800 ", "price: 50 ")
  • aggregate — aggregate statistics including minimum, maximum, and average (e.g. "average: 50 ")
  • facet — facet name (e.g. "profit" and "expenditures")
  • name — name (e.g. "x-axis titled price")
  • children — children's names (e.g. at the root, "axes titled date and price")
  • quantile — quantile of current data (e.g. "quartile: 1 ")
  • depth — depth of element in tree (e.g. "depth: 3 ")
